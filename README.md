README
This repository contains three Python scripts for hypergraph-based spreading dynamics research.

Code Overview
Code 1: hypergraph_noloop_more_seed.py
This script generates higher-order hypergraphs, which are used to model complex interactions involving more than two nodes simultaneously.

Code 2: 
SAIR_on_hypergraph_choice_E_jianhua.py
SAIR_on_hypergraph_choice_E_jianhua_selfp_control.py
This script provides the foundational implementation of the SAIR model on hypergraphs. It serves as the core framework for simulating epidemic or information spreading processes.

Code 3: parallel_SAIR_spreading_nodes_all_repeat_bili_zifenzu_more.py 
This script conducts spreading experiments based on the hypergraph SAIR model. It runs simulations to analyze propagation dynamics under different parameter settings.

Code 4: 
parallel_SAIR_spreading_nodes_all_control_moretime_V1.py
parallel_SAIR_spreading_nodes_all_control_moretime_V2.py
parallel_SAIR_spreading_nodes_all_control_moretime_V3.py
This script evaluates behavioral, hyperedge-based, and node-based intervention strategies based on the hypergraph SAIR model.
